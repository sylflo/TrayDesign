Overview

We chose to use layers help to differentiate between the different kinds of tasks performed by the components, making it easier to create a design that supports reusability of components.

Tray Design is comprised of three layers, Desktop Layer, Tray Layer and Gadget (content) layer, each layer performing a specific type of task.

DESKTOP
LAYER

The desktop layer is not movable, it performs our theme pictures as a background.

TRAY
LAYER

The Tray layer contains numbers of trays, each tray organizes information into a container that groups related content. Tray layers always use white color, and they can only move vertically on the desktop layer.

Balance and Rhythm

Separate text from images

Most of fonts are not free to use. We've been avoiding to add text content in images when designing banners or product pictures. Instead, we chose to simply overlay the text over images by coding. other than to avoid causing copyright issues, we can ensure that the text contents are displayed properly even on smaller screens and at dierent resolutions and also to make them easier to edit when needed by using this technique.

Composition

There are two ways to layout the images and text contents.

When thereâ€™s a gap between the image and text contents, the width of the gap should always be equal or greater than 30px.

When text content is overlapping image, the width of the overlap area should not be exceed half width of the text content itself.
